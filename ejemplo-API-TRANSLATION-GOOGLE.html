API TRANSLATION DE CLOUD


Dentro del mundo de las APIs Google nos ofrece una variada cantidad de ellas unas de pago y otras gratis dentro de las APIs de pago se encuentra la 'API TRANSLATION DE CLOUD'.

Características:

  • Precio mínimo por traducir menos de un millón de caracteres es de 20 dólares al mes
  • La velocidad de traducción es de 30 milisegundos dependiendo de la velocidad de banda ancha del cliente que realiza la solicitud.
  • Detección automática de idioma

Ejemplo:

El ejemplo de uso de esta 'API TRANSLATION DE CLOUD' se basa en aprovechar una de sus características es el uso de REST que permite el envió de una página HTML y mediante la API obtenemos de vuelta la página traducida al idioma seleccionado, por favor escoge un idioma y la pagina automáticamente estará traducida en dicho idioma adelante has la prueba:
